# 클로드 코드 기본 세팅

1. 가격

- 구독 요금제: 추천
- API 과금형 요금제: 비추
- Vertex AI 등 클라우드 기반 요금제: 기업

2. 세팅

- esc: 명령 취소
- esc 2번 누르면: 대화 히스토리를 보고 위로 올라가서 거기에서 다시 대화를 시작할 수 있습니다.
- claude --continue: 진행중이던 대화 다시 이어서 하기
- /resume: 예전 대화 다시 선택하기
- /usage: 현재 사용량 점검
- /model: 모델 선택, 가능하면 가격이 낮은 것을 기본으로 사용하자

3. 중요한 기능

- /init: CLAUDE.md 파일을 생성한다.
- /context: 현재 컨텍스트가 얼마나 쌓여있는지 알려줍니다.
- /compact: 지금까지 나눈 컨텍스트 요약 후 문서로 남긴다.
- /clear: 컨텍스트 클리어
- /rewind: 대화 복구

4. 기타

- permission
- rule
- hook
- subagent
- claude.md 여러개 만들어서 쓰기
